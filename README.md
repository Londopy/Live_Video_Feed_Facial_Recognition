# Live_Video_Feed_Facial_Recognition
A live video feed that scans your face. Prints out you're gender, age, race, &amp; facial expression.

---
## Getting Started

### Installation
###### It's too easy

1. Download the .zip from @Releases

2. Un-Zip that ting

3. Run the installer

4. Enjoy the program!
